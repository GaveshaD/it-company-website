# IT Company Website

A beginner-friendly IT company website built with Nuxt.

## Features

- Home page
- About page
- Services page
- Responsive design
- Reusable Vue components
- Navigation bar
- Footer

## Tech Stack

- Nuxt
- Vue
- HTML
- CSS
- Git
- GitHub

## Project Goal

This project is created to learn web development step by step by building a real website.

## Run Locally

```bash
npm install
npm run dev
```
